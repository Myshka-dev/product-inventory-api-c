# PRODUCT INVENTORY API - Subgroup C

## overview
in-memory REST API for managing product inventorywith full CRUD operations

## Setup Instructions

### prerequisites
- Node.js
- npm

### Instalation
- on ternimal, hit : git clone https://github.com/Myshka-dev/product-inventory-api-c.git

- hit : npm install (to install the dependencies)

### environmental Variables
- create a ".env" file , add PORT = YOUR DESIRED PORT NUMBER eg : PORT = 3001

### Running the Server
- npm run dev (development with auto-reload)  OR
- npm start
